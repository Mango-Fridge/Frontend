# mango

냉장고 물품 관리 및 '전국통합식품영양성분정보표준데이터'API를 활용한 토이 프로젝트

## VERSION 1.0.2 (2025-02-28)
 - sign_in_with_apple 에서 이메일을 못가져오는 현상 수정
 - shared_preferences를 활용한 로컬 저장 (email, platform)
 - go_router 적용
 - (임시) 약관 동의 View
