# mango

냉장고 물품 관리 및 '전국통합식품영양성분정보표준데이터'API를 활용한 토이 프로젝트

## VERSION 1.0.6 (2025-03-13)
 - 메인화면
   - 물품 추가 Gesture 적용으로 키보드 내리기
   - 검색해서 물품 추가 데이터 전달 (Search -> Add)
 - 요리
   - 요리 화면 View 및 Focus 적용
