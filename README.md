# mango

냉장고 물품 관리 및 '전국통합식품영양성분정보표준데이터'API를 활용한 토이 프로젝트

## VERSION 1.1.2 (2025-04-14)
 - 메인화면
   - 물품을 선택했을때, 중분류명도 함께 작성/수정 되게 변경
 - 그룹
   - 그룹 로드 및 나가기 관련 수정
 - 요리
   - 요리 리스트 유무에 따른 추가 버튼 배치 변경
   - 검색 TextField "X" 버튼을 누르면 포커스 해제
   - 수량값이 이전에 작성된것과 동일하게 적용되는 현상 수정
